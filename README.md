# Books App with Google's BookAPI

Used frameworks/libs:

  * *ReactJS*
  * *SASS (scss)*
  * *Axios*
  * *Material-ui* with [***google font icons***](https://fonts.google.com/icons)
  * *Typescript*
  

## Disclaimer
You must first change your api key in ***.env.template*** file. 
  
Then start app with..
```
$ npm start
```
